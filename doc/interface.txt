使用方式例子：/login?user_name=abc&passwd=hello
一、登陆类接口：
1)用户登陆
/login
参数：
user_name：[必填]用户名
passwd:[必填]密码

2)用户退出
/login/logout
参数：无

二、注册类接口：
1)用户注册
/register
参数：
user_name：[必填]用户名
passwd:[必填]密码
email:[必填]邮箱

2)用户激活
/register/active_account
参数：
user_name：[必填]用户名
active_code：[必填]激活码

三、文章类接口：
1)获取文章列表(按热度排序)
/article/get_list
参数：
type_id：[必填]列表类型(0:推荐, 1:心理, 2:真相, 3:一图读懂, 4:肿瘤, 5:慢病, 6:营养, 7:母婴)
num：[选填]用于分页，每页多少个，默认为10个
page：[选填]用于分页，第几页，默认为第1页

2)获取文章列表(按时间排序)
/article/get_latest_list
参数：
num：[选填]用于分页，每页多少个，默认为10个
page：[选填]用于分页，第几页，默认为第1页
detail：[选填]用于判断是否显示详细信息，默认为1(0:只包括文章ID、标题和创建时间, 1:包括显示列表所需要的信息)

3)获取单篇文章的信息(显示文章列表时所需的信息，无作者信息)
/article/get_info_list
参数：
article_id：[必填]文章ID

4)获取单篇文章的信息(显示单篇文章时所需的信息，无摘要信息)
/article/get_info_detail
参数：
article_id：[必填]文章ID

5)获取文章内容
/article/get_content
参数：
article_id：[必填]文章ID

6)搜索
/article/search
参数：
key_word：[必填]关键词
num：[选填]用于分页，每页多少个，默认为10个
page：[选填]用于分页，第几页，默认为第1页

7)发布文章
/article/publish
参数：
title：[必填]文章标题
author：[必填]作者名
author_desp：[必填]作者描述
type_id：[必填]列表类型(0:推荐, 1:心理, 2:真相, 3:一图读懂, 4:肿瘤, 5:慢病, 6:营养, 7:母婴)
content：[必填]文章内容(html格式)
图片上传参数：
coverimg：封面图的form name
headimg：作者头像的form name

8)修改文章
/article/modify
参数：
article_id：[必填]文章ID
title：[必填]文章标题
author：[必填]作者名
author_desp：[必填]作者描述
type_id：[必填]列表类型(0:推荐, 1:心理, 2:真相, 3:一图读懂, 4:肿瘤, 5:慢病, 6:营养, 7:母婴)
content：[必填]文章内容(html格式)
图片上传Html参数：
coverimg：封面图的form name
headimg：作者头像的form name

9)上传文章插图
/article/upload_content_img
图片上传Html参数：
articleimg

10)文章点赞
/article/like
参数：
article_id：[必填]文章ID

11)删除文章
/article/remove
参数：
article_id：[必填]文章ID

四、评论类接口：
1)添加评论
/comment/add
参数：
article_id：[必填]文章ID
content：[必填]评论内容
parent_id：[选填]父评论ID

2)删除评论(子评论也会被删除)
/comment/remove
参数：
comment_id：[必填]评论ID

3)评论点赞
/comment/like
参数：
comment_id：[必填]评论ID

4)评论踩
/comment/unlike
参数：
comment_id：[必填]评论ID

5)获取评论列表
/comment/get_article_comments
参数：
article_id：[必填]文章ID

五、收藏类接口
1)添加收藏
/collection/add
参数：
article_id：[必填]文章ID

2)删除收藏
/collection/remove
参数：
article_id：[必填]文章ID

3)获取我的收藏列表
/collection/get_my_collection
参数：
num：[选填]用于分页，每页多少个，默认为10个
page：[选填]用于分页，第几页，默认为第1页

六、轮播类接口
1)发布轮播
/carousel/publish
参数：
carousel_ids：[必填]轮播ID列表，多个轮播ID使用下划线连接，例如: 332_356_367

2)添加轮播
/carousel/add
参数：
article_id：[必填]文章ID
图片上传form name: carouselimg

4)获取当前生效的轮播
/carousel/get_effect_list
参数：无

5)获取已经添加的轮播列表
/carousel/get_list
参数：
num：[选填]用于分页，每页多少个，默认为10个
page：[选填]用于分页，第几页，默认为第1页
